Concept Definitions

Computer Network
is a group of computer systems and computer hardware devices linked together to be able to communicate and share things among the computers.

Data Transmission
The process of sending digital or analog data to one or more computing, network, communication or electronic device.

Client-Server Architecture
is a network architecture; Servers manage drives, printers, and network servers. Clients are the PC's where users run applications; Clients rely on servers for resources such as files and and devices.

HTTP
This means Hyper Text Transfer Protocol used to define how messages are formatted and transmitted. It also tells web servers what to do when it recieves various commands.

Hypermedia
is basically an extension of HTTP where it allows you to use things such as audio and video.

World Wide Web
an information system that allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.

Internet
a global computer network which provides a bunch of information to connect networks.

TCP/IP
Means a "Transmission Control Protocol/ Internet Protocol." This controls the connection of computer systems to the internet.

Virtualization
the creation of virtual resource such as a server, desktop, operating system, file, storage or network.

Kernel Virtual Machine
A process which seperates the software application and computer hardware to distinct between the two
