Definition List

HTML Terms

  • HTML:

    A system that can change text files to customize things on a web page.

  • HTML elements:

    The text or element that is inbetween the HTML tags.

  • HTML tag:

    Commands that can change text or elements or start a line.

  • HTML bracket:

    The beginning symbols of an HTML element(they are angled brackets)

  • HTML attribute

    Extra bits of info to put in an HTMLL tag to specify its' commands even further.

  • HTML comment

    This is to make comments in the file, but the person viewing the web page won't be able to see the comment, only the owner.

  • HTML object element

    This defines an embedded object within an HTML document.

  • HTML special character

    Characters that are special.
