Computer Operating Systems

Operating System
The main software of a computer that allow you to perform simple tasks

Batch Processing
Is when a program does a large amount of work in the background without the user having to do anything, like collecting usage data.

Timesharing
This is when mutliple people share the same computer, as in they can all access the machine at the sametime.

Multitasking
This means the computer can execute more than 1 task at the sametime.

Multiprocessing
Computers can run more than 2 programs or instructions at the sametime.

Computer Architecture
Cache, CPU, Hardware, Machine Instruction Cycle, Main Memory, Network, Peripheral, RAM

Operating System Structure
There are 2 ways am operating system can run; 1st: Human User to Application Software to Operating System to Other System Software to Hardware; 2nd: Human User to Operating System to Hardware

System Boot
It boots up big programs, such as the operating system.

CISC vs RISC
Complex Instruction Set Computer(CISC): single instructions can execute low level operations; Reduced Instruction Set Computer(RISC): is a computer that uses the CPU to make the instructions easier to read to make the processing quick
